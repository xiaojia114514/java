# 课程目标达成情况分析系统开发计划

## 一、框架分析与调整

### 保留的核心模块

1. **用户认证与授权**：保留若依框架的用户登录、权限管理功能，确保系统安全
2. **基础框架**：Spring Boot 3、MyBatis-Plus、Vue 3 等核心技术栈
3. **数据字典**：用于存储课程类型、目标类型等基础数据
4. **文件上传**：支持Excel成绩导入、报告导出等功能
5. **日志系统**：记录系统操作日志，便于问题排查

### 保留的额外模块

1. **通知公告**：用于向教师发布系统通知、课程相关信息等
2. **代码生成器**：用于快速生成基础CRUD代码，提高开发效率

### 移除的模块

1. **监控模块**：druid监控、服务器监控等
2. **系统配置**：简化配置管理
3. **岗位管理**：简化用户管理
4. **部门管理**：简化组织架构

## 二、系统模块设计

### 1. 课程管理模块

- 课程基本信息管理（课程名称、代码、学分、学期等）
- 课程负责人管理
- 课程列表查询与筛选

### 2. 教学目标管理模块

- 教学目标的增删改查
- 目标层级管理（一级目标、二级目标）
- 目标描述与权重设置
- 目标与课程关联

### 3. 试卷管理模块

- 试卷基本信息管理（试卷名称、考试时间、总分等）
- 试题管理（选择题、判断题、简答题等）
- 试题与教学目标关联（设置关联度权重）
- 试卷与课程关联

### 4. 学生管理模块

- 学生基本信息管理（学号、姓名、班级等）
- 学生与课程关联
- 学生列表查询与筛选

### 5. 成绩管理模块

- Excel成绩导入功能
- 成绩录入与编辑
- 成绩查询与统计
- 成绩与试卷关联

### 6. 达成度分析模块

- 基于试题-目标关联计算达成度
- 达成度统计与分析
- 达成度可视化展示
- 历史数据分析

### 7. 报告生成模块

- 自动生成课程目标达成情况分析报告
- 报告模板管理
- 报告导出（Word、PDF）
- 报告历史记录

### 8. LLM集成模块

- LLM API配置管理
- 报告表述优化
- 智能分析建议
- 自然语言交互

## 四、功能实现计划

### 1. 教学目标管理功能

- **实现目标**：支持课程教学目标的完整生命周期管理
- **核心功能**：
  - 目标的增删改查操作
  - 目标层级关系管理
  - 目标权重设置
  - 目标与课程关联
- **技术实现**：
  - 后端：Spring Boot + MyBatis-Plus
  - 前端：Vue 3 + Element Plus
  - 数据校验：JSR-303

### 2. 试卷管理和试题与目标关联功能

- **实现目标**：支持试卷和试题的管理，以及试题与教学目标的关联
- **核心功能**：
  - 试卷基本信息管理
  - 试题的增删改查
  - 试题与目标的关联设置（权重）
  - 试卷与课程关联
- **技术实现**：
  - 后端：Spring Boot + MyBatis-Plus
  - 前端：Vue 3 + Element Plus
  - 表单验证：自定义验证规则

### 3. 学生成绩导入和管理功能

- **实现目标**：支持学生成绩的批量导入和管理
- **核心功能**：
  - Excel成绩导入
  - 成绩录入与编辑
  - 成绩查询与统计
  - 成绩与试卷关联
- **技术实现**：
  - 后端：Spring Boot + EasyExcel
  - 前端：Vue 3 + Element Plus + XLSX
  - 数据验证：Excel数据校验

### 4. 达成度计算算法

- **实现目标**：基于试题-目标关联计算教学目标达成度
- **核心算法**：
  1. 计算每个试题的得分率
  2. 根据试题与目标的关联权重，计算每个目标的得分
  3. 根据目标权重，计算课程整体达成度
- **技术实现**：
  - 后端：Java算法实现
  - 数据存储：MySQL
  - 性能优化：批量计算

### 5. 报告自动生成功能

- **实现目标**：自动生成课程目标达成情况分析报告
- **核心功能**：
  - 报告模板管理
  - 报告内容自动生成
  - 报告导出（Word、PDF）
  - 报告历史记录
- **技术实现**：
  - 后端：Spring Boot + POI + iText
  - 前端：Vue 3 + Element Plus
  - 模板引擎：FreeMarker

### 6. 集成LLM用于报告表述优化

- **实现目标**：利用LLM优化报告表述，提供智能分析建议
- **核心功能**：
  - LLM API配置管理
  - 报告表述优化
  - 智能分析建议
  - 自然语言交互
- **技术实现**：
  - 后端：Spring Boot + OpenAI API
  - 前端：Vue 3 + Element Plus
  - 接口封装：RESTful API

### 7. 前端界面开发和测试

- **实现目标**：开发直观、易用的前端界面
- **核心功能**：
  - 响应式布局
  - 数据可视化
  - 交互优化
  - 权限控制
- **技术实现**：
  - 前端框架：Vue 3
  - UI组件库：Element Plus
  - 图表库：ECharts
  - 测试工具：Jest + Cypress

### 8. 系统测试和部署

- **实现目标**：确保系统稳定运行，方便部署
- **核心功能**：
  - 单元测试
  - 集成测试
  - 性能测试
  - 部署配置
- **技术实现**：
  - 测试框架：JUnit 5 + Mockito
  - 部署工具：Docker + Docker Compose
  - CI/CD：GitHub Actions

## 六、技术选型
